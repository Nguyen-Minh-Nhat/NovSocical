User
-

Email string
PassWord string
FirstName string
LastName string
AvatarPath char
BirthDate date
CreateAt date

Follow
-
Follower PK int FK >- User.UserID
Following PK int FK >- User.UserID

Post
-
PostID PK int AUTOINCREMENT
UserID int FK >- User.UserID
PostText string NULL
PostImagePath string NULL
isEdit boolean=false
CreateAt date

Comment
-
CommentID PK int AUTOINCREMENT

ReplyOf int=-1

CommentImagePath string NULL
isEdit boolean=false
